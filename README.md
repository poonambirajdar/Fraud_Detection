# Fraud_Detection
Fraud Detection System using Machine Learning with Python, including data preprocessing, model building, and visualization

📊 Dataset:- 
    https://kaggle.com/datasets/poonambirajdar8870/fraud-detection

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Features
- Data preprocessing
- Handling missing values
- Model training (Random Forest)
- Evaluation (Accuracy, Precision, Recall)
